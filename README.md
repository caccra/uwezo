# Uwezo Security Services — Landing Page

Official marketing website for **Uwezo Security Services Limited**, a licensed tactical security provider based in Kampala, Uganda.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Homepage — hero, services overview, sectors summary |
| `about.html` | Company background and values |
| `services.html` | Full services listing |
| `sectors.html` | Industries served |
| `contact.html` | Inquiry and contact form |
| `service-guards.html` | Armed & Unarmed Guards |
| `service-vip.html` | VIP Protection |
| `service-cit.html` | Cash in Transit (CIT) Logistics |
| `service-cctv.html` | CCTV Surveillance |
| `service-corporate.html` | Corporate Security |
| `service-residential.html` | Residential Protection |
| `service-event.html` | Event Security Management |
| `service-investigation.html` | Investigations |
| `service-training.html` | Security Training |

## Tech Stack

- Plain HTML5, CSS3, and vanilla JavaScript — no build tools required
- Google Fonts (Inter, Outfit)
- Responsive design with mobile-first breakpoints

## Running Locally

Open `index.html` directly in a browser, or serve with any static file server:

```bash
npx serve .
```
